FitPulse – Health Anomaly Detection from Fitness Devices
Project Description

FitPulse is a data-driven project that focuses on detecting health anomalies using data collected from fitness devices such as smartwatches and fitness bands. The goal is to analyze time-series health data and identify unusual patterns that may indicate abnormal physiological behavior.

This repository contains Milestone-1, which covers data understanding, preprocessing, visualization, and basic anomaly detection techniques.

Objectives

Understand fitness device data

Clean and preprocess raw health data

Perform exploratory data analysis (EDA)

Identify abnormal patterns using basic anomaly detection methods

Technologies Used

Programming Language: Python

Libraries: pandas, NumPy, Matplotlib, Seaborn, scikit-learn

Platform: Google Colab / Jupyter Notebook

Dataset

The dataset consists of fitness-related parameters such as:

Heart rate

Step count

Calories burned

Activity duration

Timestamped records

(Data is stored in CSV format) collected by kaggle 

Methodology

Data Collection

Load fitness device data from CSV files

Data Preprocessing

Handle missing values

Remove duplicates

Normalize and format time-series data

Exploratory Data Analysis

Visualize trends and distributions

Identify outliers using plots

Anomaly Detection

Apply statistical and ML-based methods

Detect abnormal health patterns

git clone https://github.com/Deepu9981/FitPulse-Health-Anomaly-Detection-from-Fitness-Devices.git
